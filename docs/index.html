Enhancement One: Software Design & Engineering

Object-Oriented Refactoring

Refactored the monolithic procedural script into a modular Object-Oriented application. This enhancement introduced the Room, Player, and GameMap classes to improve maintainability and encapsulate state.

Enhancement Two: Algorithms & Data Structures

Procedural Generation & Optimization

Implemented a Modified Prim's Algorithm to procedurally generate a unique dungeon layout every game. Integrated Breadth-First Search (BFS) for map validation and optimized inventory lookups from O(n) to O(1).

Enhancement Three: Databases

Data Persistence with MongoDB

Integrated a NoSQL database to replace static data structures. Created a robust Save/Load system that serializes the complex graph of the procedurally generated map, ensuring true data persistence.
